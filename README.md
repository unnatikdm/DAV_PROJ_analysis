# Nifty 50 Stock Analysis 📊

## Overview
This project analyzes Nifty 50 stock data using Python and Tableau.

## Features
- Data cleaning and preprocessing
- Moving averages (MA50, MA100)
- Volatility analysis
- Trend detection (Bullish/Bearish)

## Tech Stack
- Python (Pandas)
- Tableau

## Workflow
Raw Data → Cleaning → Feature Engineering → Dashboard
